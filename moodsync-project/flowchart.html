🎨→🎵 MoodSync Workflow

1
👤 User Input & URL Processing
User enters Pinterest board URL → Frontend validates URL → Backend receives request
📝 URL validation and sanitization
🔗 Pinterest board URL extraction
⚡ Rate limiting check (3 req/min per user)
⬇️
2
📸 Image Extraction & Vision API Analysis
Backend scrapes Pinterest board → Extracts 5 images → Google Vision API analyzes each image
🕷️ Web scraping of Pinterest board
🖼️ Image URL extraction (5 images max)
🎨 Google Vision API analysis per image
🏷️ Object detection with confidence scores
👥 Face detection and expression analysis
🌈 Color palette extraction
📝 Text recognition (OCR)
⬇️
3
🧠 Mood Detection & Aggregation
For each image: Color analysis → Mood mapping → Object analysis → Face analysis → Text analysis
🎨 Color temperature analysis (warm/cool/bright)
😊 Mood mapping (passionate/calm/energetic/melancholic)
🏃 Activity detection (workout/relaxation/social)
🏠 Setting analysis (indoor/outdoor/urban/rural)
📊 Confidence scoring for each mood
🔄 Aggregation across all 5 images
⬇️
4
🤖 AI-Powered Music Recommendations
Claude AI receives visual analysis → Generates sophisticated music recommendations
📋 Claude AI prompt creation with analysis data
🎵 Genre recommendations (3-5 specific genres)
⚡ Energy level determination (low/medium/high)
🎼 Tempo range specification (BPM)
😊 Mood characteristics mapping
🔍 Spotify search terms generation
🎛️ Audio features targeting
🔄 Fallback to rule-based system if AI fails
⬇️
5
🎵 Spotify Integration & Playlist Creation
Backend uses AI recommendations → Searches Spotify API → Creates personalized playlist
🔍 Spotify API search with AI-generated terms
🎛️ Audio feature filtering
🎲 Track randomization and selection (20 tracks)
📝 Playlist creation with mood-appropriate name
➕ Adding tracks to user's Spotify account
🔗 Playlist URL generation
⬇️
6
🖥️ Frontend Display & User Experience
Frontend receives results → Displays animated analysis → Shows playlist with Spotify player
📊 Animated analysis visualization
😊 Mood breakdown with confidence scores
🤖 AI-generated recommendations display
🎵 Generated playlist with Spotify player
🔄 Option to create new playlists
📱 Responsive design for all devices

Frontend React.js, CSS3, HTML5
Backend Node.js, Express.js
AI/ML Claude AI, Google Vision API
Music Spotify Web API
Social Pinterest API
